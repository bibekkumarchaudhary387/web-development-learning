Steps to Create an HTML File (Ordered)

  1. Open a text editor (like Notepad or VS Code).
  2. Write your HTML code.
  3. Save the file with a .html extension.
  4. Open the file in a web browser.

Web Terms (Description List)

HTML
HyperText Markup Language used for creating web pages.
HTTP
HyperText Transfer Protocol used for transmitting data over the web.
